Project Overview

Objective

This project focuses on the design and manufacturing of a traceable 3D printed phone component using industrial additive manufacturing techniques.

The system integrates:

  • CAD modelling
  • Industrial FDM printing
  • QR code traceability
  • Manufacturing optimisation
  • PFMEA analysis
  • Quality validation

CAD Design & Development

The phone case assembly was designed using CAD modelling principles with dimensional tolerances optimized for FDM printing. Snap-fit geometry and QR integration were carefully developed to ensure functionality and manufacturability.

3D Printing & Manufacturing

Manufacturing Process

Multiple print iterations were tested to optimise:

  • Surface quality
  • Dimensional accuracy
  • Print stability
  • QR readability
  • Assembly tolerance

Both Ultimaker S5 and Bambu Lab P1S systems were evaluated.

Print Parameter Optimisation

PFMEA Analysis

Process Failure Mode and Effects Analysis (PFMEA) was conducted to identify manufacturing risks, evaluate severity levels, and implement corrective actions.

Digital Traceability System

Each printed component includes an integrated QR code linking to digital manufacturing records, print parameters, inspection results, and validation documentation.

Prototype Gallery

Industrial Validation

FESTO CP Lab Integration

The manufactured component was tested and validated within an industrial FESTO CP Lab environment.

Validation included:

  • Assembly fit verification
  • Dimensional consistency
  • QR readability
  • Mechanical compatibility
  • Repeatability assessment

Future Scope

Smart Manufacturing

Integration with Industry 4.0 production systems.

Automated Quality Control

AI-driven defect detection and process optimisation.

Real-time digital monitoring of manufactured components.
