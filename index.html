Project Overview
This project investigates the development of 3D printed phone case assemblies using FDM additive manufacturing. The work integrates: manufacturing optimisation, digital traceability, FESTO CP Lab automation, and Tecnomatix Plant Simulation to explore modern Industry 4.0 manufacturing systems.
Additive Manufacturing Journey
Over 50 print iterations were manufactured using Ultimaker S5, Ultimaker S3, and Bambu Lab P1S printers. Multiple parameters including: print speed, infill, support structures, brim width, and snap-fit tolerances were systematically optimised to achieve dimensional accuracy and successful FESTO assembly.
FESTO CP Lab Validation
Final assemblies were validated through the FESTO CP Lab automated assembly line, including: Measuring, Muscle Press, Tunnel Furnace, and Workpiece Discharge stations. The Muscle Press station became the critical validation point for snap-fit optimisation.
Tecnomatix Plant Simulation
A digital production model was developed using Tecnomatix Plant Simulation to analyse throughput, bottlenecks, printer utilisation, and production scalability. Experiment Manager studies demonstrated that 3D printing capacity — not worker count — was the primary system bottleneck.
